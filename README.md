Production-ready SaaS landing page (28kb gzipped, zero dependencies) showcasing a premium productivity app with glassmorphism design, custom 60fps magnetic cursor, and hyper-realistic phone mockups.
# 🚀 Lumio — Focus & Flow App

> Your focus, supercharged.

Lumio is a modern, visually stunning productivity landing page designed to simulate a real-world SaaS application. It blends **deep work science with elegant UI/UX design** to help users achieve peak productivity.

## ✨ Features

### 🎯 Core Features

* 🧠 AI-Powered Flow Engine (UI simulation)
* ⏱ Smart Pomodoro Timer
* 📊 Productivity Analytics Dashboard
* 🎵 Focus Soundscapes
* 🔒 App Blocker Interface
* 🔄 Cross-device Sync UI

### 💡 UI/UX Features

* 🌌 Modern glassmorphism + gradient UI
* 🌀 Smooth scroll animations (Intersection Observer)
* 🎯 Custom animated cursor
* 📱 Fully responsive design
* 🎨 Floating gradient orbs & micro-interactions
* ⚡ Animated progress bar
* 🔥 Hover effects & transitions

### ⚙️ Functional Features

* 📩 Email validation system
* 📊 Animated stats counter
* 💳 Pricing toggle (Monthly ↔ Yearly)
* 📽 Demo modal popup
* 📱 Mobile navigation menu

## ⚙️ How It Works

1. User lands on the homepage
2. Scroll-based animations reveal sections
3. Interactive elements:

   * Navigation scroll
   * Modal popup
   * Pricing toggle
   * Email validation
4. UI simulates a real SaaS product experience
